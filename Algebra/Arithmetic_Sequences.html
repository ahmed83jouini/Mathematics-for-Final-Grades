Arithmetic Sequences

In an arithmetic sequence, the difference between consecutive terms is constant.

For example, the sequence 2, 4, 6, 8 is arithmetic because the difference is always 2.

To find the nth term of an arithmetic sequence, use the formula:

a_n = a_1 + (n - 1) d

where:

  • a_n is the nth term.
  • a_1 is the first term.
  • d is the common difference.

Understanding arithmetic sequences can help in various problem-solving scenarios.
