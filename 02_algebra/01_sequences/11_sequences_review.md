  - h3: "1. طرق توليد المتتالية والاتجاه"
  - parags:
      - rtl: "تُعرّف المتتالية إما بعبارة <b>الحد العام</b> $u_n = f(n)$ أو بعلاقة <b>تراجعية</b> $u_{n+1} = f(u_n)$."
      - rtl: "لدراسة اتجاه التغير نستخدم أحد المعايير الثلاثة:"
      - rtl: "• حساب الفرق: $u_{n+1} - u_n$ ومقارنته بالصفر."
      - rtl: "• الحساب النسبي (للحدود الموجبة): $\\frac{u_{n+1}}{u_n}$ ومقارنته بالواحد."
      - rtl: "• دراسة تغيرات الدالة المرفقة $f$ (خاص بالحد العام فقط)."

  - h3: "2. المتتاليات الشهيرة (الحسابية والهندسية)"
  - parags:
      - rtl: "الجدول أدناه يختصر القوانين الحاسمة التي يحتاجها التلميذ:"

  - table:

      rows:
          - ["التعريف", "$u_{n+1} = u_n + r$", "$u_{n+1} = u_n \\times q$"]
          - ["الحد العام", "$u_n = u_p + (n-p)r$", "$u_n = u_p \\times q^{n-p}$"]
          - ["الوسط", "$a + c = 2b$", "$a \\times c = b^2$"]

  - h3: "3. التقارب والخصائص السلوكية"
  - parags:
      - rtl: "• تكون المتتالية <b>متقاربة</b> إذا كانت نهايتها حقيقية $\\lim\\limits_{n \\to +\\infty} u_n = L$، وغير ذلك فهي <b>متباعدة</b>."
      - rtl: "• <b>مبرهنة المحدودية والاتجاه:</b> كل متتالية متزايدة ومحدودة من الأعلى هي متقارية. وكل متناقصة ومحدودة من الأدنى هي متقارية."
      - rtl: "• <b>المتتاليات المتجاورة:</b> (الدرس السابق) إحداهما متزايدة والأخرى متناقصة وفرقهما يؤول إلى الصفر، وتحصران النهاية بينهما."
