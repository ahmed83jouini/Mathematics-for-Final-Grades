elements:
  - h3: "1. ميكانيكية حساب المميز (The Discriminant)"
  - parags:
      - rtl: "لحل المعادلة التربيعية النموذجية $az^2 + bz + c = 0$ حيث المعاملات أعداد حقيقية حتماً ($a, b, c \\in \\mathbb{R}$) مع ($a \\neq 0$)، نعتمد ميكانيكياً على حساب المميز الفرزي الشهير دلتا:"
      - center: "$\\Delta = b^2 - 4ac$"

  - h3: "2. الفرز الهيكلي للحالات الثلاث للمميز"
  - parags:
      - rtl: "تتوزع طبيعة الحلول الجبرية في المجموعات العددية بناءً على الإشارة الصافية للمميز الميكانيكي كالتالي:"
      - rtl: "• <b>إذا كان $\\Delta > 0$:</b> المعادلة تملك حلين حقيقيين متمايزين صافيين (تماماً كما في الحقل المألوف $\\mathbb{R}$)."
      - rtl: "• <b>إذا كان $\\Delta = 0$:</b> المعادلة تملك حلاً حقيقياً واحداً مضاعفاً تسويته الجبرية هي: $z_0 = -\\frac{b}{2a}$."
      - rtl: "• <b>إذا كان $\\Delta < 0$:</b> هنا تظهر القوة الاختراقية للمجموعة $\\mathbb{C}$؛ حيث تصبح المعادلة مقبولة الحل وتملك حتمياً حلين مركبين مترافقين هندسياً."

  - h3: "3. خوارزمية التعامل مع المميز السالب"
  - parags:
      - rtl: "عندما تسفر الحسابات عن مميز سالب تماماً ($\\Delta < 0$)، نقوم بحيلة جبرية ميكانيكية لتسييله عبر تعويض الإشارة السالبة بالرمز $i^2$ وتحت القيمة المطلقة:"
      - center: "$\\Delta = -|\\Delta| = i^2 |\\Delta|$"
      - rtl: "وبذلك يصبح الجذران التربيعيان للمميز داخل الحقل التخيلي هما: $i\\sqrt{|\\Delta|}$ و $-i\\sqrt{|\\Delta|}$."
      - rtl: "وتكون صياغة الحلين المركبين المترافقين النهائيين كالتالي:"
      - center: "$z_1 = \\frac{-b - i\\sqrt{|\\Delta|}}{2a}, \\quad z_2 = \\frac{-b + i\\sqrt{|\\Delta|}}{2a}$"

  - h4: "مثال تطبيقي بيداغوجي: حل المعادلة $z^2 - 2z + 5 = 0$"
  - parags:
      - rtl: "1. <b>خطوة الفرز:</b> نحسب المميز: $\\Delta = (-2)^2 - 4(1)(5) = 4 - 20 = -16$."
      - rtl: "2. <b>التطهير التخيلي:</b> بما أن $\\Delta < 0$، نعيد صياغته ميكانيكياً على الشكل: $\\Delta = 16i^2$، ومنه جذر المميز هو $4i$."
      - rtl: "3. <b>الحل الأول الصافي:</b> $z_1 = \\frac{2 - 4i}{2} = 1 - 2i$."
      - rtl: "4. <b>الحل الثاني الصافي:</b> $z_2 = \\frac{2 + 4i}{2} = 1 + 2i$ وهو بالتأكيد المرافق المباشر للحل الأول ($\\bar{z}_1$)."
