  - h3: "1. المفهوم الميكانيكي"
    id: "vibration-concept"

  - parags:
      - rtl: "هذه المعادلة تصف نظاماً لا يضيع الطاقة؛ حيث تتناسب 'القوة المسترجعة' (المتمثلة في $y''$) طردياً مع البعد عن المركز ($y$). الإشارة الموجبة لـ $\\omega^2$ تضمن أن الحركة دورية واهتزازية وليست أسية."

  - h3: "2. المبرهنة والحل العام"
    id: "theorem-second-order"

  - parags:
      - rtl: "الحلول العامة للمعادلة التفاضلية $y'' + \\omega^2y = 0$ (حيث $\\omega \\neq 0$) هي مجموعة الدوال المعرفة على $\\mathbb{R}$ بـ:"
      - center: "$ f(x) = A \\cos(\\omega x) + B \\sin(\\omega x) $"
      - rtl: "أو بالصيغة الجيبية المدمجة:"
      - center: "$ f(x) = y_{max} \\cos(\\omega x + \\phi) $"
      - rtl: "حيث $A$ و $B$ (أو $y_{max}$ و $\\phi$) هي ثوابت تحددها الشروط الابتدائية (الموضع والسرعة عند اللحظة 0)."

  - h3: "3. بروتوكول الحل"
    id: "protocol-vibration"
  - tableLines:
      dir: rtl
      lines:
        - - th: "المرحلة"
          - th: "الإجراء"
        - - td: "1. استخراج النبض"
          - td: "حساب قيمة $\\omega$ (جذر المعامل المضروب في $y$)."
        - - td: "2. الصياغة العامة"
          - td: "كتابة الحل بدلالة الجيب والجيوب تمام: $A \\cos(\\omega x) + B \\sin(\\omega x)$."
        - - td: "3. توظيف الشروط"
          - td: "نحتاج هنا لشرطين (قيمة الدالة وقيمة مشتقتها) لإيجاد $A$ و $B$."



  - h3: "4. مثال من واقع النواس"
    id: "vibration-example"
  - parags:
      - rtl: "<b>المسألة:</b> حل المعادلة $y'' + 4y = 0$ مع الشروط $y(0) = 1$ و $y'(0) = 0$."
      - rtl: "<b>تحديد المعامل:</b> $\\omega^2 = 4 \\implies \\omega = 2$."
      - rtl: "<b>الحل العام:</b> $y(x) = A \\cos(2x) + B \\sin(2x)$."
      - rtl: "<b>تطبيق الشرط الأول:</b> $y(0) = 1 \\implies A \\cos(0) + B \\sin(0) = 1 \\implies A = 1$."
      - rtl: "<b>تطبيق الشرط الثاني:</b> نشتق $y'(x) = -2A \\sin(2x) + 2B \\cos(2x)$. نعوض $y'(0) = 0 \\implies -2(1)(0) + 2B(1) = 0 \\implies B = 0$."
      - rtl: "<b>الحل الخاص:</b> $f(x) = \\cos(2x)$."

        - rtl: "$\\omega$ يسمى في الفيزياء 'النبض الذاتي'، ومنه نحسب الدور $T = \\frac{2\\pi}{\\omega}$."
