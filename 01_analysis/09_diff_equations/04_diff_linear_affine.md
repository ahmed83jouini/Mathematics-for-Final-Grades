elements:
  - h3: "1. المفهوم التقني"
    id: "affine-concept"

  - parags:
      - rtl: "هذه المعادلة هي تطوير للمبحث السابق، حيث أضفنا ثابتاً $b$ يمثل 'قوة خارجية' أو 'ثابتاً مضافاً'. الحل هنا يسعى دائماً نحو قيمة حدية توازن بين التأثير الأسي والثابت $b$."

  - h3: "2. المبرهنة والحل العام"
    id: "theorem-affine"

  - parags:
      - rtl: "الحلول العامة للمعادلة التفاضلية $y' = ay + b$ (حيث $a \\neq 0$) هي مجموعة الدوال المعرفة على $\\mathbb{R}$ بـ:"
      - center: "$ f(x) = C \\cdot e^{ax} - \\frac{b}{a} $"
      - rtl: "لاحظ أن الحل يتكون من جزأين: جزء أسي متغير، وجزء ثابت وهو $-\\frac{b}{a}$."

  - h3: "3. بروتوكول الحل"
    id: "protocol-affine"
  - tableLines:
      dir: rtl
      lines:
        - - th: "المرحلة"
          - th: "الإجراء"
        - - td: "1. الاستخراج"
          - td: "تحديد قيمتي $a$ و $b$ بدقة مع مراعاة الإشارات."
        - - td: "2. الصياغة"
          - td: "كتابة الحل العام مباشرة: $y = C e^{ax} - \\frac{b}{a}$."
        - - td: "3. التخصيص"
          - td: "استخدام الشرط الابتدائي لإيجاد القيمة الدقيقة للثابت $C$."

  - h3: "4. مثال من واقع المختبر"
    id: "lab-example"
  - parags:
      - rtl: "<b>المسألة:</b> حل المعادلة $y' = -2y + 4$ مع الشرط $y(0) = 1$."
      - rtl: "<b>تحديد المعاملات:</b> $a = -2$ و $b = 4$."
      - rtl: "<b>الحساب:</b> $-\\frac{b}{a} = -\\frac{4}{-2} = 2$."
      - rtl: "<b>الحل العام:</b> $y = C e^{-2x} + 2$."
      - rtl: "<b>إيجاد C:</b> عند $x=0$ يكون $y=1$ $\\implies 1 = C e^{0} + 2 \\implies 1 = C + 2 \\implies C = -1$."
      - rtl: "<b>الحل الخاص:</b> $f(x) = -e^{-2x} + 2$."



  - NB:
      title: "للعلم!:"
      type: "warning"
      NBs:
        - rtl: "في الفيزياء، القيمة $-\\frac{b}{a}$ تمثل غالباً 'النظام الدائم' (القيمة التي تستقر عندها الدالة بعد زمن طويل)."
        - rtl: "انتبه لإشارة الكسر $-\\frac{b}{a}$؛ فكثير من الطلبة يخطئون في إشارة الناقص الأصلية في القانون."
