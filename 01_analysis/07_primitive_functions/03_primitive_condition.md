elements:
  - h3: "1. مبرهنة الوحدانية"
    id: "uniqueness-theorem"

  - parags:
      - rtl: "لتكن $f$ دالة مستمرة على مجال $I$. من أجل كل عدد حقيقي $x_0$ من $I$ وكل عدد حقيقي $y_0$، توجد<strong> دالة أصلية وحيدة</strong> $F$ للدالة $f$ تحقق الشرط:"
      - center: "$ F(x_0) = y_0 $"

  - h3: "2. كيف نحدد الثابت $c$؟"
    id: "finding-c"

  - parags:
      - rtl: "في المبحث السابق، قلنا إن مجموعة، أو عائلة الدوال الأصلية هي: $F(x) + c$. لتحديد دالة وحيدة، نحتاج لـ 'نقطة تثبيت' $(x_0, y_0)$. نقوم بتعويض هذه النقطة في العبارة العامة لنحل معادلة مجهولها هو $c$."

  - h3: "3. مثال تطبيقي (خطوات الورقة)"
    id: "step-by-step-example"
  - parags:
      - rtl: "<strong>المسألة:</strong> أوجد الدالة الأصلية $F$ للدالة:"
      - center: "$f(x) = 2x + 1$"
      - rtl: "والتي تنعدم عند $1$ (أي $F(1) = 0$)."
      - rtl: "<strong>الخطوة 1:</strong> نجد الشكل العام للدوال الأصلية:"
      - center: "$F(x) = x^2 + x + c$."
      - rtl: "<strong>الخطوة 2:</strong> نطبق الشرط $F(1) = 0$."
      - center: "$ (1)^2 + (1) + c = 0 \\implies 2 + c = 0 \\implies c = -2 $"
      - rtl: "<strong>الخطوة 3:</strong> نكتب الدالة الوحيدة المطلوبة:"
      - center: "$F(x) = x^2 + x - 2$."

  - h3: "4. التفسير الهندسي (تثبيت المنحنى)"
    id: "geometric-fixing"
  - parags:
      - rtl: "من بين كل المنحنيات المتوازية التي رأيناها سابقاً، هناك منحنى واحد فقط يمر عبر النقطة $(x_0, y_0)$. تحديد القيمة المعلومة هو بمثابة وضع 'مسمار' يثبت عائلة المنحنيات في مكان واحد لا تتزحزح عنه."
